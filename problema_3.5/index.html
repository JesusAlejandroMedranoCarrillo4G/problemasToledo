Problema 3.5

👥 Persona de Menor Edad

Ingresa el nombre y la edad de tres personas. El algoritmo determinará cuál de ellas es la de menor edad.

Pseudocódigo:
LEER nombre1, edad1, nombre2, edad2, nombre3, edad3
menor ← nombre1, edadMenor ← edad1
SI edad2 < edadMenor → menor ← nombre2, edadMenor ← edad2
SI edad3 < edadMenor → menor ← nombre3, edadMenor ← edad3
ESCRIBIR menor, edadMenor
